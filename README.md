# Govee

[![Add To Installation](https://marketplace.signalrgb.com/resources/add-extension-256.png 'Add to My SignalRGB Installation')](signalrgb://addon/install?url=https://gitlab.com/mgv101/Govee_H6609)

## Getting started
A fork from the original Govee addon https://gitlab.com/signalrgb/Govee to fix issues with the H6609 Govee Gaming Light Strip G1.

MAKE SURE TO DISABLE THE ORIGINAL GOVEE PLUG-IN!!

## Known Issues
- Only works if your LED strip is running clockwise around the monitor (when looking from the front of the monitor).
- Will most likely stop all other Govee devices from functioning properly.

## Installation
Click the button above and allow signalrgb to install this extension when prompted.
